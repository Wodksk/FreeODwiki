# 李云齐论   
By @[SalviaSWC](https://x.com/SalviaSWC)

<i>本文档中的某些事实是作者根据个人记忆写的，需要验证，因此放在观点讨论区。建议在验证工作完成后，有关人员将本文档转移到“[社会学](/文档/社会学/home.md)”目录下。</i>

李云齐是一个提供各类服务的<strong>私人性质</strong>的团队发言人。有关服务包括但不限于：“禁毒叙事”为基础的精神活性物质科普信息、精神活性物质检测服务、精神活性物质检测套件、避免有关部门的精神活性物质筛查的“排毒”产品、涉毒案件辩护律师。

早在2025年，李云齐就已经在社交媒体上公开从事涉OD社群的事务。当时，他以后土百科的名义，拍摄了一集纪录片，邀请的受访者包括沐雨。

然而，在[2026年4月的风波](文档/od.md)中，他因疑似与OD社群的某些劣迹斑斑的成员勾结，插手社群内部私人事务、威胁向警方提供社群成员名单、利用社交媒体，传播有损社群荣誉和社群安全的误导信息等危害严重、影响恶劣的行为而引发社群成员的广泛关注。编写此文档的意义在于，向社群提供他的一些公开信息，并以此为依据展开讨论，达成澄清有关李云齐的错误观念的目的。
