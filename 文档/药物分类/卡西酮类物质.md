# 卡西酮类物质

[◀返回](./home.md)

| [![](/文件/Substituted_cathinone.png)](/文件/Substituted_cathinone.png) |
| ------------------------------------------------------------------- |
| 卡西酮分子的取代结构。                                                         |

**取代卡西酮类物质**（也称**卡西酮**或**合成卡西酮**）是一类以[卡西酮](/药物/卡西酮.md)结构为基础的精神活性化合物。这个类别由卡西酮的一切衍生化合物组成；这些化合物是通过在卡西酮核心结构中，用取代基替换一个或多个氢原子而形成的。例子包括[MDPV](/药物/MDPV.md)、[4-MMC](/药物/4-MMC.md)、[MDMC](/药物/MDMC.md)和[α-PVP](/药物/α-PVP.md)。

卡西酮是恰特草（*Catha edullis*）中的活性成分，人们会咀嚼它的叶片来获得兴奋作用呢。卡西酮在结构上与[苯丙胺](/药物/苯丙胺.md)关系很近，唯一的区别是它在β位碳上多了一个酮基。

## 化学

卡西酮类物质是一类以[卡西酮](/药物/卡西酮.md)分子为基础的化学类别。卡西酮类物质是[苯丙胺类物质](/文档/药物分类/苯丙胺类物质.md)的一个子类别。它们共享苯丙胺的核心结构，但会因为在苯丙胺结构的β位碳上存在一个酮基取代，而与[取代苯丙胺类物质](/文档/药物分类/苯丙胺类物质.md)区分开来。因此，有些卡西酮，例如[βk-MDMA](/药物/MDMC.md)，只是相较于某种常见取代苯丙胺多了一个β-酮基取代，于是就会在该取代苯丙胺名称前加上“beta-keto”/“βk-”前缀来命名。
需要注意的是，虽然它们严格来说并不属于卡西酮类物质，因为其结构中缺少[卡西酮](/药物/卡西酮.md)所具有的α位碳甲基，但仍有一类精神活性化合物，是在各种[取代苯乙胺类物质](/文档/药物分类/苯乙胺类物质.md)的基础上进行β-酮基取代而得到的；其中最著名的例子是[βk-2C-B](/药物/βk-2C-B.md)。

具有共情作用和[兴奋剂](/文档/药物分类/兴奋剂.md)作用的卡西酮类药物，主要通过其作为[多巴胺](/文档/多巴胺.md)[再摄取抑制剂](/文档/神经递质再摄取抑制剂.md)的活性而起效。对这一组受体的[激动作用](/文档/受体激动剂.md)会使突触后神经元的放电速率增加，从而在使用者身上触发[思维加速](/药效/思维加速.md)与[刺激](/药效/刺激.md)这两类效应。
