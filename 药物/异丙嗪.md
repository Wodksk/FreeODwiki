**异丙嗪**（常见销售为**非那根**或**勒吉甘**）是一种[吩噻嗪类](/药物分类/ )的[抑制剂](/药物分类/抑制剂.md)。它与可待因一起，是“瘦汁”中两种活性成分之一呢。它作为[抗组胺药](/文档/抗组胺药.md)发挥作用。

异丙嗪由法国化学家在1940年代中期开发。今天，它在许多国家以多种品牌名称上市。

[主观效应](/药效/主观效应索引.md)包括[肌肉松弛](/药效/肌肉松弛.md)、[恶心抑制](/药效/恶心抑制.md)以及强烈的[镇静](/药效/镇静.md)。它还能减轻晕动症，并具有抗胆碱特性。

异丙嗪已被证明具有良好的[催眠药](/文档/催眠药.md)效果，有时用于此目的呢。

如果使用这种物质，强烈建议采用伤害减少措施哦。

异丙嗪在1940年代中期由罗纳-普朗克实验室的一组科学家从吩噻嗪和苯海拉明的二胺侧链合成而成。它以前被用作抗精神病药，虽然现在一般不再用于此目的。它大约具有氯丙嗪抗精神病强度的1/10。

它在大约1990年左右在休斯顿说唱和嘻哈圈流行起来，这种趋势在2015至2016年左右再次出现。通常，它与阿片类药物[可待因](/药物/可待因.md)联合制成“紫水”。紫水通常包含冰、雪碧或柑橘苏打水、异丙嗪/可待因止咳糖浆，有时还有欢乐牧场糖来提升风味呢。

药理

异丙嗪是组胺H1受体的直接拮抗剂，而[苯海拉明](/药物/苯海拉明.md)是反向激动剂。它也是毒蕈碱乙酰胆碱受体的拮抗剂，这可能是高剂量时出现镇静和谵妄的原因。它阻断多巴胺D2受体，但作用较弱。

主观效应
------------------

### **躯体效应** ![Child.png](/文件/Child.png)

  + **[镇静](/药效/镇静.md)** - 异丙嗪引起明显的镇静和整体缓慢感，通常伴随不想移动的感觉，这种效应通常比苯海拉明或多西拉敏强得多，由于其高抗胆碱、H1拮抗和弱D2拮抗活性，一般被认为是最具镇静作用的抗组胺药之一呢。
+ **[肌肉松弛](/药效/肌肉松弛.md)**
    - **[躯体欣快感](/药效/躯体欣快感.md)** - 这可能导致全身放松的非常愉悦感觉，像温暖的凝胶一样。它类似于但不同于[阿片类](/药物分类/阿片类药物.md)的躯体欣快感。
+ **[心率增快](/药效/心率增快.md)** 和 **[血压升高](/药效/血压升高.md)** - 毒蕈碱乙酰胆碱受体的抗胆碱活性引起这些效应。
+ **[心律异常](/药效/心律异常.md)** - 这种效应极为罕见。
+ **[恶心抑制](/药效/恶心抑制.md)** - 异丙嗪用于治疗晕动症或恶心。
+ **[口干](/药效/口干.md)** - 这是这种物质的常见副作用。
+ **[排尿困难](/药效/排尿困难.md)** - 这种副作用不常见。

### **认知效应** ![User.png](/文件/User.png)

+ **[焦虑抑制](/药效/焦虑抑制.md)** - 通常，异丙嗪作为[抗焦虑](/文档/抗抑郁药.md)剂发挥作用。
+ **[焦虑](/药效/焦虑.md)** - 异丙嗪偶尔可能因谵妄引起偏执和焦虑，虽然比苯海拉明、多西拉敏和氯苯那敏少见。
+ **[欣快感](/药效/认知欣快.md)** - 通常，异丙嗪引起极小的欣快感，通常仅在焦虑个体中出现，甚至可能引起烦躁，然而当以低剂量与可待因、羟考酮或氢可酮等阿片类联合使用时，它能有效增强欣快效应并允许较低的阿片剂量。
+ **[情感抑制](/药效/情感抑制.md)** - 这种效应较轻，由于轻微的弱抗精神病效应发生，不过远弱于氯丙嗪或丙氯拉嗪。
+ **[清醒错觉](/药效/谵妄.md)** - 这是尽管有明显相反证据（如严重认知损害和无法完全与他人沟通），却错误相信自己完全清醒的信念。
+ **[困倦](/药效/困倦.md)** - 身体疲惫和疲倦感是异丙嗪常见且明显的效应。因此，这种物质用于治疗失眠和异常睡眠周期呢。
+ **[谵妄](/药效/谵妄.md)** 和 **[思维混乱](/药效/思维混乱.md)** - 在低至中等剂量下，异丙嗪呈现无害的困惑或谵妄感。在高剂量和过量时，它可能（但很少）严重到患者或用户经历兴奋性谵妄，即严重激动和混乱的状态。

### 致死剂量

异丙嗪在人类中的最低已发表毒性剂量（[口服](/文档/给药途径.md#口服)）为3.5 mg/kg。这意味着体重70 kg的人在245 mg时可能出现毒性迹象。异丙嗪在小鼠中的[LD50](/文档/治疗指数.md)（[口服](/文档/给药途径.md#口服)）为255 mg/kg。如果应用于人类，这表明体重70 kg的50%的人在摄入17.85克异丙嗪后会死亡。

### 耐受性和成瘾潜力

异丙嗪没有成瘾性。

***警告：*** *许多单独使用相当安全的的精神活性物质，当与某些其他物质联合时，可能突然变得危险甚至危及生命。以下列表提供了一些已知的危险联用（虽然不保证包含全部）。*

*始终进行独立研究（例如[Google](https://www.google.com)、[DuckDuckGo](https://www.duckduckgo.com)、[PubMed](https://pubmed.ncbi.nlm.nih.gov/)）以确保两种或多种物质的组合安全食用。部分列出的联用来源于[TripSit](https://combo.tripsit.me)。* 由于异丙嗪广泛的药理学，它有许多联用。根据[Drugs.com](https://www.drugs.com)的联用检查器，异丙嗪已知与超过1000种其他处方药和非处方药相互作用。

* **[抑制剂](/药物分类/抑制剂.md)** (*[1,4-丁二醇](/药物/1,4-丁二醇.md)*, [2-甲基-2-丁醇](/药物/2M2B.md), [酒精](/药物/酒精.md), [巴比妥类物质](/药物分类/巴比妥类物质.md), [GHB](/药物/GHB.md)/[GBL](/药物/GBL.md), [甲喹酮](/药物/甲喹酮.md), [阿片类药物](/药物分类/阿片类药物.md)*) - 这种组合可能导致危险甚至致命的[呼吸抑制](/药效/呼吸抑制.md)水平。这些物质相互增强[肌肉松弛](/药效/肌肉松弛.md)、[镇静](/药效/镇静.md)和[失忆](/药效/失忆.md)，并可能在高剂量下导致意外意识丧失。无意识期间呕吐风险增加，并可能因窒息导致死亡。如果发生，用户应尝试以[恢复体位](/文档/恢复体位.md)入睡，或让朋友将其移至该体位。
* **[抗多巴胺药](/文档/多巴胺.md)** - 由于异丙嗪也阻断多巴胺受体，其他药物和物质这样做会增加急性或迟发性运动障碍、神经阻滞剂恶性综合征或帕金森综合征的发生几率。
* **[抗胆碱药](/文档/抗组胺药.md)** - 异丙嗪与抗胆碱药（或抗毒蕈碱药）联合可能导致乙酰胆碱阻断增加，可能危险并伴随心血管效应以及谵妄。
* **[兴奋剂](/药物分类/兴奋剂.md)** - 由于异丙嗪的兴奋性心脏效应，与兴奋剂联合会增加异常心律、严重心动过速或心脏病发作以及其他心血管事件的风险。
