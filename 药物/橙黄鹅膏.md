# 橙黄鹅膏

[◀返回](index.md)

![橙黄鹅膏](/文件/Amanita_sp._2010-10-31.jpg)

橙黄鹅膏

***橙黄鹅膏***（异名 *Amanita mappa*），常被称为**假死帽鹅膏**或**香橼鹅膏**，是一种担子菌类蘑菇，也是鹅膏属中的众多物种之一呢。它会在夏季和秋季生长于硅质土壤中。其菌盖呈淡黄色，有时也会是白色，菌柄、菌环和菌托均为白色。虽然它并不致命，但不可食用，而且常常会和致命的毒鹅膏混淆哦。

按干重计算，它的菌盖中含有0.8%的[蟾毒色胺](/药物/蟾毒色胺.md)，菌柄中含有1.5%，基部膨大部中含有0.065%。

## 外观

这种蘑菇具有肉质的淡黄色菌盖，有时也会呈白色，直径为4–10厘米（1.5–4英寸），表面覆有不规则斑块。菌褶和菌肉为白色。6–8厘米（2.5–3英寸）高的菌柄基部有较大的菌托，菌柄上带有明显的菌环。它常常会和近缘的毒鹅膏混淆，这也正是其名称的由来呢。

## 法律状态

除非法律明确将蟾毒色胺列为非法，否则这种蘑菇本身通常不受禁止。由于它不含蝇蕈醇，在大多数法律体系下，它不会被归类为具有精神活性的鹅膏菌物种。

* **澳大利亚——** 根据澳大利亚联邦政府《刑法条例》，蟾毒色胺被列为附表I受控物质。它同时也被列入《毒物标准》（2015年10月）的附表9物质。1964年《毒物法》将附表9药物界定为：“可能被滥用或误用，其制造、持有、销售或使用除经首席执行官批准、用于医学或科学研究，或用于分析、教学或培训目的外，应依法禁止的物质。”根据1981年《滥用药物法》，6.0克（0.21盎司）被认定为足以进入法院审理的数量，而2.0克（0.071盎司）则被视为具有销售和供应意图。
* **英国——** 在英国，蟾毒色胺依据1971年《滥用药物法》被列为A级药物。
* **美国——** 在美国联邦层面，蟾毒色胺（DEA Drug Code 7403）受缉毒局作为附表I药物监管，因此购买、持有和销售均属非法。
